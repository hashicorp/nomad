# `contains` Function

`contains` determines whether a given list or set contains a given single value
as one of its elements.

```hcl
contains(list, value)
```

## Examples

```shell-session
> contains(["a", "b", "c"], "a")
true
> contains(["a", "b", "c"], "d")
false
```
