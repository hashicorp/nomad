# `lower` Function

`lower` converts all cased letters in the given string to lowercase.

## Examples

```shell-session
> lower("HELLO")
hello
> lower("АЛЛО!")
алло!
```

This function uses Unicode's definition of letters and of upper- and lowercase.

## Related Functions

- [`upper`](/nomad/docs/reference/hcl2/functions/string/upper) converts letters in a string to _uppercase_.
- [`title`](/nomad/docs/reference/hcl2/functions/string/title) converts the first letter of each word in a string to uppercase.
