# Secure Nomad

This section provides guidance on securing Nomad. Configure a Nomad cluster for ACLs, bootstrap the ACL system, author your first policy, and grant a token based on it. Configure SSO options such as OIDC and JWT. Use workload identity to federate access to AWS and GCP.
