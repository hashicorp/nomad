# `trimsuffix` Function

`trimsuffix` removes the specified suffix from the end of the given string.

## Examples

```shell-session
> trimsuffix("helloworld", "world")
hello
```

## Related Functions

- [`trim`](/nomad/docs/reference/hcl2/functions/string/trim) removes characters at the start and end of a string.
- [`trimprefix`](/nomad/docs/reference/hcl2/functions/string/trimprefix) removes a word from the start of a string.
- [`trimspace`](/nomad/docs/reference/hcl2/functions/string/trimspace) removes all types of whitespace from
  both the start and the end of a string.
