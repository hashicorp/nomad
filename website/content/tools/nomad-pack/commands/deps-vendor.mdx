# Nomad-Pack Deps vendor

Command: `nomad-pack deps vendor`

Vendor dependencies for a pack.

@include "tools/nomad-pack/commands/deps-vendor_desc.mdx"

## Usage

Usage: `nomad-pack deps vendor`


	Vendor dependencies for a pack in the current directory.

Examples:
	# Create a new registry named "my-new-registry" in the current directory.
	nomad-pack generate registry my-new-registry

#### Vendoring Options

- `-path=<string>` - Full path to the pack which contains dependencies to be
				    vendored. All the dependencies will then be downloaded
				    into a 'deps/' subdirectory of that path.
- `-timeout=<int>` - Timeout (in seconds) for downloading dependencies.
