# `nomad windows service uninstall` command reference

The `windows service uninstall` command removes the nomad Windows service.

## Usage

```plaintext
nomad windows service uninstall
```

The `windows service uninstall` command will stop the nomad service if
it is currently running, deregister the service with the Windows eventlog,
and remove the Windows service. The installed nomad binary, as well as
the data and configuration directories, will not be removed.
